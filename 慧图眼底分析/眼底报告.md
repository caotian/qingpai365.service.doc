## 生成患者编号

```
post /AIFundus/patient
```
>请求参数 appId，shopId,cuntomerId，



>>返回 patientNo：患者编号

## 根据患者编号获取报告JSON

```
get /AIFundus/report
```
>请求参数 patientNo:患者编号 

>返回 reportData
