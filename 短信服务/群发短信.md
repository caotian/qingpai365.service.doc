## 分页查询所有短信任务

```
GET /sms/smsTask
```
## 分页查询短信列表
```
GET /sms
```

## 获取短信发送条数

```
GET /sms/total
```
> 返回  allTotal：总条数； todayTotal：今日短信竖向


## 删除短信任务

```
DELETE /sms/smsTask/{id}
```
## 修改短信任务

```
Put /sms/smsTask
```

## 发送短信

```
Put /sms/smsTask
```
